# Sagarr

**Sagarr** is a self-hosted, AI-powered media recommendation engine designed to fit seamlessly into the "Arr" stack. It bridges the gap between your Plex watch history (via Tautulli) and your media acquisition (Overseerr), providing personalized, dynamic recommendations.

## Features
*   **AI-Driven Discovery:** Generates creative categories based on your mood and history.
*   **Seamless Integration:** Works with Plex, Tautulli, and Overseerr.
*   **Privacy Focused:** Self-hosted with local data storage.

## Architecture
*   **Backend:** Python (FastAPI)
*   **Frontend:** React
*   **Database:** SQLite
